bigbadselect

A jQuery plugin for creating minimalistic custom select(s).

Download

Examples

Simple

Requirements

Currently, it needs jQuery greater than 1.9.1

How it works

A select like this

<select>
  <option value="m">Male</option>
  <option value="f">Female</option>
</select>

generates html like this

<div class="bigbadselect">
  <div class="bigbadselect-current-container">Male</div>
  <div class="bigbadselect-options-container">
    <div class="bigbadselect-option selected" data-val="m">Male</div>
    <div class="bigbadselect-option" data-val="f">Female</div>
  </div>
</div>
