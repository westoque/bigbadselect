bigbadselect

A jQuery plugin for creating minimalistic custom select(s). The plugin is very simplistic, and designed for people who want to do their own styling. Comes with its own basic css that can be easily modified.

Download

Demo

How to Use

Just include bigbadselect.css and bigbadselect.js to your project. Requires jQuery > 1.9.1

<link rel="stylesheet" type="text/css" href="bigbadselect.css">
<script src="bigbadselect.js" type="text/javascript"></script>

How it works

A select like this

<select>
  <option value="m">Male</option>
  <option value="f">Female</option>
</select>

bigbadselect(ed) like this

$('select').bigbadselect();

generates html like this

<div class="bigbadselect">
  <div class="bigbadselect-current-container">Male</div>
  <div class="bigbadselect-options-container">
    <div class="bigbadselect-option selected" data-val="m">Male</div>
    <div class="bigbadselect-option" data-val="f">Female</div>
  </div>
</div>

Options

currentTmpl - The template for the currently showed selected option. You have to return an html string that will be rendered. Passed are two arguments, the value and text of the option.

var currentTmpl = function(val, text) {
  return '<div>' + text + '(' + val + ')</div>';
};
$('select').bigbadselect({currentTmpl: currentTmpl});

optionTmpl - The template for an option in the options list. Like the currentTmpl, you need to return an html string that will be rendered. Also passed two arguments, namely, the val and text of the option.

var optionTmpl = function(val, text) {
  return '<div>Value: ' + val + '</div>'
};
$('select').bigbadselect({optionTmpl: optionTmpl});
